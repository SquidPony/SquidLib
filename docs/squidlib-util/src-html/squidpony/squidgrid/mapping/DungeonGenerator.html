017/**
018 * The primary way to create a more-complete dungeon, layering different effects and modifications on top of
019 * a DungeonBoneGen's dungeon or another dungeon without such effects. Also ensures only connected regions of the map
020 * are used by filling unreachable areas with walls, and can find far-apart staircase positions if generate() is used or
021 * can keep existing staircases in a map if generateRespectingStairs() is used.
022 * <br>
023 * The main technique for using this is simple: Construct a DungeonGenerator, usually with the desired width and height,
024 * then call any feature adding methods that you want in the dungeon, like addWater(), addTraps, addGrass(), or
025 * addDoors(). Some of these take different parameters, like addDoors() which need to know if it should check openings
026 * that are two cells wide to add a door and a wall to, or whether it should only add doors to single-cell openings.
027 * Then call generate() to get a char[][] with the desired dungeon map, using a fixed repertoire of chars to represent
028 * the different features. After calling generate(), you can safely get the values from the stairsUp and stairsDown
029 * fields, which are Coords that should be a long distance from each other but connected in the dungeon. You may want
030 * to change those to staircase characters, but there's no requirement to do anything with them. It's recommended that
031 * you keep the resulting char[][] maps in some collection that can be saved, since DungeonGenerator only stores a
032 * temporary copy of the most recently-generated map. The DungeonUtility field of this class, utility, is a convenient
033 * way of accessing the non-static methods in that class, such as randomFloor(), without needing to create another
034 * DungeonUtility (this class creates one, so you don't have to).
035 * <br>
036 * Previews for the kinds of dungeon this generates, given a certain argument to generate():
037 * <ul>
038 *     <li>Using TilesetType.DEFAULT_DUNGEON (text, click "Raw", may need to zoom out): https://gist.github.com/tommyettinger/a3bd413b903f2e103541</li>
039 *     <li>Using TilesetType.DEFAULT_DUNGEON (graphical, scroll down and to the right): http://tommyettinger.github.io/home/PixVoxel/dungeon/dungeon.html</li>
040 *     <li>Using SerpentMapGenerator.generate()  (text, click "Raw", may need to zoom out): https://gist.github.com/tommyettinger/93b47048fc8a209a9712</li>
041 * </ul>
042 * <br>
043 * As of March 6, 2016, the algorithm this uses to place water and grass was swapped for a more precise version. You no
044 * longer need to give this 150% in addWater or addGrass to effectively produce 100% water or grass, and this should be
045 * no more than 1.1% different from the percentage you request for any effects. If you need to reproduce dungeons with
046 * the same seed and get the same (imprecise) results as before this change, it's probably not possible unless you save
047 * the previously-generated char[][] dungeons, since several other things may have changed as well.
048 * @see squidpony.squidgrid.mapping.DungeonUtility this class exposes a DungeonUtility member; DungeonUtility also has many useful static methods
049 *
050 * @author Eben Howard - http://squidpony.com - howard@squidpony.com
051 * @author Tommy Ettinger - https://github.com/tommyettinger
052 */
053public class DungeonGenerator {

179    /**
180     * Make a DungeonGenerator with the given height, width, and RNG. Use this if you want to seed the RNG. If width or
181     * height is greater than 256, then this will expand the Coord pool from its 256x256 default so it stores a
182     * reference to each Coord that might be used in the creation of the dungeon (if width and height are 300 and 300,
183     * the Coord pool will be 300x300; if width and height are 500 and 100, the Coord pool will be 500x256 because it
184     * won't shrink below the default size of 256x256).
185     * @param width The width of the dungeon in cells
186     * @param height The height of the dungeon in cells
187     * @param rng The RNG to use for all purposes in this class; if it is a StatefulRNG, then it will be used as-is,
188     *            but if it is not a StatefulRNG, a new StatefulRNG will be used, randomly seeded by this parameter
189     */
190    public DungeonGenerator(int width, int height, RNG rng)
191    {
192        Coord.expandPoolTo(width, height);
193        this.rng = (rng instanceof StatefulRNG) ? (StatefulRNG) rng : new StatefulRNG(rng.nextLong());
194        gen = new DungeonBoneGen(this.rng);
195        utility = new DungeonUtility(this.rng);
196        rebuildSeed = this.rng.getState();
197        this.height = height;
198        this.width = width;
199        fx = new EnumMap<>(FillEffect.class);
200    }

219    /**
220     * Turns the majority of the given percentage of floor cells into water cells, represented by '~'. Water will be
221     * clustered into a random number of pools, with more appearing if needed to fill the percentage.
222     * Each pool will have randomized volume that should fill or get very close to filling the requested
223     * percentage, unless the pools encounter too much tight space. If this DungeonGenerator previously had addWater
224     * called, the latest call will take precedence. No islands will be placed with this variant, but the edge of the
225     * water will be shallow, represented by ','.
226     * @param percentage the percentage of floor cells to fill with water
227     * @return this DungeonGenerator; can be chained
228     */
229    public DungeonGenerator addWater(int percentage)
230    {
231        if(percentage < 0) percentage = 0;
232        if(percentage > 100) percentage = 100;
233        if(fx.containsKey(FillEffect.WATER)) fx.remove(FillEffect.WATER);
234        fx.put(FillEffect.WATER, percentage);
235        return this;
236    }
237    /**
238     * Turns the majority of the given percentage of floor cells into water cells, represented by '~'. Water will be
239     * clustered into a random number of pools, with more appearing if needed to fill the percentage. Each pool will
240     * have randomized volume that should fill or get very close to filling the requested percentage,
241     * unless the pools encounter too much tight space. If this DungeonGenerator previously had addWater called, the
242     * latest call will take precedence. If islandSpacing is greater than 1, then this will place islands of floor, '.',
243     * surrounded by shallow water, ',', at about the specified distance with Euclidean measurement.
244     * @param percentage the percentage of floor cells to fill with water
245     * @param islandSpacing if greater than 1, islands will be placed randomly this many cells apart.
246     * @return this DungeonGenerator; can be chained
247     */
248    public DungeonGenerator addWater(int percentage, int islandSpacing)
249    {
250        if(percentage < 0) percentage = 0;
251        if(percentage > 100) percentage = 100;
252        if(fx.containsKey(FillEffect.WATER)) fx.remove(FillEffect.WATER);

254        if(fx.containsKey(FillEffect.ISLANDS)) fx.remove(FillEffect.ISLANDS);
255        if(islandSpacing > 1)
256            fx.put(FillEffect.ISLANDS, islandSpacing);
257        return this;
258    }

291    /**
292     * Turns the given percentage of viable doorways into doors, represented by '+' for doors that allow travel along
293     * the x-axis and '/' for doors that allow travel along the y-axis. If doubleDoors is true,
294     * 2-cell-wide openings will be considered viable doorways and will fill one cell with a wall, the other a door.
295     * If this DungeonGenerator previously had addDoors called, the latest call will take precedence.
296     * @param percentage the percentage of valid openings to corridors to fill with doors; should be between 10 and
297     *                   20 if you want doors to appear more than a few times, but not fill every possible opening.
298     * @param doubleDoors true if you want two-cell-wide openings to receive a door and a wall; false if only
299     *                    one-cell-wide openings should receive doors. Usually, this should be true.
300     * @return this DungeonGenerator; can be chained
301     */
302    public DungeonGenerator addDoors(int percentage, boolean doubleDoors)
303    {
304        if(percentage < 0) percentage = 0;
305        if(percentage > 100) percentage = 100;
306        if(doubleDoors) percentage *= -1;
307        if(fx.containsKey(FillEffect.DOORS)) fx.remove(FillEffect.DOORS);
308        fx.put(FillEffect.DOORS, percentage);
309        return this;
310    }

339    protected OrderedSet<Coord> removeAdjacent(OrderedSet<Coord> coll, Coord pt)
340    {
341        for(Coord temp : new Coord[]{Coord.get(pt.x + 1, pt.y), Coord.get(pt.x - 1, pt.y),
342                Coord.get(pt.x, pt.y + 1), Coord.get(pt.x, pt.y - 1)})
343        {
344            if(coll.contains(temp) && !(temp.x == pt.x && temp.y == pt.y))
345                coll.remove(temp);
346        }
347
348        return coll;
349    }
350    protected OrderedSet<Coord> removeAdjacent(OrderedSet<Coord> coll, Coord pt1, Coord pt2)
351    {
352
353        for(Coord temp : new Coord[]{Coord.get(pt1.x + 1, pt1.y), Coord.get(pt1.x - 1, pt1.y),
354                Coord.get(pt1.x, pt1.y + 1), Coord.get(pt1.x, pt1.y - 1),
355                Coord.get(pt2.x + 1, pt2.y), Coord.get(pt2.x - 1, pt2.y),
356                Coord.get(pt2.x, pt2.y + 1), Coord.get(pt2.x, pt2.y - 1),})
357        {
358            if(coll.contains(temp) && !(temp.x == pt1.x && temp.y == pt1.y) && !(temp.x == pt2.x && temp.y == pt2.y))
359                coll.remove(temp);
360        }
361
362        return coll;
363    }
364    protected OrderedSet<Coord> viableDoorways(boolean doubleDoors, char[][] map)
365    {
366        OrderedSet<Coord> doors = new OrderedSet<>();
367        OrderedSet<Coord> blocked = new OrderedSet<>(4);
368        DijkstraMap dm = new DijkstraMap(map, DijkstraMap.Measurement.EUCLIDEAN);
369        for(int x = 1; x < map.length - 1; x++) {
370            for (int y = 1; y < map[x].length - 1; y++) {
371                if(map[x][y] == '#')
372                    continue;
373                if (doubleDoors) {
374                    if (x >= map.length - 2 || y >= map[x].length - 2)
375                        continue;
376                    else {
377                        if (map[x+1][y] != '#' &&
378                                map[x + 2][y] == '#' && map[x - 1][y] == '#'
379                                && map[x][y + 1] != '#' && map[x][y - 1] != '#'
380                                && map[x+1][y + 1] != '#' && map[x+1][y - 1] != '#') {
381                            if (map[x + 2][y + 1] != '#' || map[x - 1][y + 1] != '#' || map[x + 2][y - 1] != '#' || map[x - 1][y - 1] != '#') {
382                                dm.resetMap();
383                                dm.clearGoals();
384                                dm.setGoal(x, y+1);
385                                blocked.clear();
386                                blocked.add(Coord.get(x, y));
387                                blocked.add(Coord.get(x + 1, y));
388                                if(dm.partialScan(16, blocked)[x][y-1] < DijkstraMap.FLOOR)
389                                    continue;
390                                doors.add(Coord.get(x, y));
391                                doors.add(Coord.get(x + 1, y));
392                                doors = removeAdjacent(doors, Coord.get(x, y), Coord.get(x + 1, y));
393                                continue;
394                            }
395                        } else if (map[x][y+1] != '#' &&
396                                map[x][y + 2] == '#' && map[x][y - 1] == '#'
397                                && map[x + 1][y] != '#' && map[x - 1][y] != '#'
398                                && map[x + 1][y+1] != '#' && map[x - 1][y+1] != '#') {
399                            if (map[x + 1][y + 2] != '#' || map[x + 1][y - 1] != '#' || map[x - 1][y + 2] != '#' || map[x - 1][y - 1] != '#') {
400                                dm.resetMap();
401                                dm.clearGoals();
402                                dm.setGoal(x+1, y);
403                                blocked.clear();
404                                blocked.add(Coord.get(x, y));
405                                blocked.add(Coord.get(x, y+1));
406                                if(dm.partialScan(16, blocked)[x-1][y] < DijkstraMap.FLOOR)
407                                    continue;
408                                doors.add(Coord.get(x, y));
409                                doors.add(Coord.get(x, y+1));
410                                doors = removeAdjacent(doors, Coord.get(x, y), Coord.get(x, y + 1));
411                                continue;
412                            }
413                        }
414                    }
415                }
416                if (map[x + 1][y] == '#' && map[x - 1][y] == '#' && map[x][y + 1] != '#' && map[x][y - 1] != '#') {
417                    if (map[x + 1][y + 1] != '#' || map[x - 1][y + 1] != '#' || map[x + 1][y - 1] != '#' || map[x - 1][y - 1] != '#') {
418                        dm.resetMap();
419                        dm.clearGoals();
420                        dm.setGoal(x, y+1);
421                        blocked.clear();
422                        blocked.add(Coord.get(x, y));
423                        if(dm.partialScan(16, blocked)[x][y-1] < DijkstraMap.FLOOR)
424                            continue;
425                        doors.add(Coord.get(x, y));
426                        doors = removeAdjacent(doors, Coord.get(x, y));
427                    }
428                } else if (map[x][y + 1] == '#' && map[x][y - 1] == '#' && map[x + 1][y] != '#' && map[x - 1][y] != '#') {
429                    if (map[x + 1][y + 1] != '#' || map[x + 1][y - 1] != '#' || map[x - 1][y + 1] != '#' || map[x - 1][y - 1] != '#') {
430                        dm.resetMap();
431                        dm.clearGoals();
432                        dm.setGoal(x+1, y);
433                        blocked.clear();
434                        blocked.add(Coord.get(x, y));
435                        if(dm.partialScan(16, blocked)[x-1][y] < DijkstraMap.FLOOR)
436                            continue;
437                        doors.add(Coord.get(x, y));
438                        doors = removeAdjacent(doors, Coord.get(x, y));
439                    }
440                }
441
442            }
443        }
444
445        return doors;
446    }

479    /**
480     * Generate a char[][] dungeon with extra features given a baseDungeon that has already been generated.
481     * Typically, you want to call generate with a TilesetType or no argument for the easiest generation; this method
482     * is meant for adding features like water and doors to existing simple maps.
483     * This uses '#' for walls, '.' for floors, '~' for deep water, ',' for shallow water, '^' for traps, '+' for doors
484     * that provide horizontal passage, and '/' for doors that provide vertical passage.
485     * Use the addDoors, addWater, addGrass, and addTraps methods of this class to request these in the generated map.
486     * Also sets the fields stairsUp and stairsDown to two randomly chosen, distant, connected, walkable cells.
487     * <br>
488     * Special behavior here: If tab characters are present in the 2D char array, they will be replaced with '.' in the
489     * final dungeon, but will also be tried first as valid staircase locations (with a high distance possible to travel
490     * away from the starting staircase). If no tab characters are present this will search for '.' floors to place
491     * stairs on, as normal. This tab-first behavior is useful in conjunction with some methods that establish a good
492     * path in an existing dungeon; an example is {@code DungeonUtility.ensurePath(dungeon, rng, '\t', '#');} then
493     * passing dungeon (which that code modifies) in as baseDungeon to this method.
494     * @param baseDungeon a pre-made dungeon consisting of '#' for walls and '.' for floors; may be modified in-place
495     * @return a char[][] dungeon
496     */
497    public char[][] generate(char[][] baseDungeon)
498    {
499        if(!seedFixed)
500        {
501            rebuildSeed = rng.getState();
502        }
503        seedFixed = false;
504        char[][] map = DungeonUtility.wallWrap(baseDungeon);
505        width = map.length;
506        height = map[0].length;
507        DijkstraMap dijkstra = new DijkstraMap(map);
508        int frustrated = 0;
509        do {
510            dijkstra.clearGoals();
511            stairsUp = utility.randomMatchingTile(map, '\t');
512            if(stairsUp == null) {
513                stairsUp = utility.randomFloor(map);
514                if (stairsUp == null) {
515                    frustrated++;
516                    continue;
517                }
518            }
519            dijkstra.setGoal(stairsUp);
520            dijkstra.scan(null);
521            frustrated++;
522        }while (dijkstra.getMappedCount() < width + height && frustrated < 8);
523        if(frustrated >= 8)
524        {
525            return generate();
526        }
527        double maxDijkstra = 0.0;
528        for (int i = 0; i < width; i++) {
529            for (int j = 0; j < height; j++) {
530                if (dijkstra.gradientMap[i][j] >= DijkstraMap.FLOOR) {
531                    map[i][j] = '#';
532                } else if (dijkstra.gradientMap[i][j] > maxDijkstra) {
533                    maxDijkstra = dijkstra.gradientMap[i][j];
534                }
535                if (map[i][j] == '\t') {
536                    map[i][j] = '.';
537                }
538            }
539        }
540        stairsDown = new GreasedRegion(dijkstra.gradientMap, maxDijkstra * 0.7,
541                DijkstraMap.FLOOR).singleRandom(rng);
542
543        return innerGenerate(map);
544    }

546    /**
547     * Generate a char[][] dungeon with extra features given a baseDungeon that has already been generated, and that
548     * already has staircases represented by greater than and less than signs.
549     * Typically, you want to call generate with a TilesetType or no argument for the easiest generation; this method
550     * is meant for adding features like water and doors to existing simple maps.
551     * This uses '#' for walls, '.' for floors, '~' for deep water, ',' for shallow water, '^' for traps, '+' for doors
552     * that provide horizontal passage, and '/' for doors that provide vertical passage.
553     * Use the addDoors, addWater, addGrass, and addTraps methods of this class to request these in the generated map.
554     * Also sets the fields stairsUp and stairsDown to null, and expects stairs to be already handled.
555     * @param baseDungeon a pre-made dungeon consisting of '#' for walls and '.' for floors, with stairs already in;
556     *                    may be modified in-place
557     * @return a char[][] dungeon
558     */
559    public char[][] generateRespectingStairs(char[][] baseDungeon) {
560        if(!seedFixed)
561        {
562            rebuildSeed = rng.getState();
563        }
564        seedFixed = false;
565        char[][] map = DungeonUtility.wallWrap(baseDungeon);
566        DijkstraMap dijkstra = new DijkstraMap(map);
567        stairsUp = null;
568        stairsDown = null;
569
570        dijkstra.clearGoals();
571        ArrayList<Coord> stairs = DungeonUtility.allMatching(map, '<', '>');
572        for (int j = 0; j < stairs.size(); j++) {
573            dijkstra.setGoal(stairs.get(j));
574        }
575        dijkstra.scan(null);
576        for (int i = 0; i < width; i++) {
577            for (int j = 0; j < height; j++) {
578                if (dijkstra.gradientMap[i][j] >= DijkstraMap.FLOOR) {
579                    map[i][j] = '#';
580                }
581            }
582        }
583        return innerGenerate(map);
584    }

589    private char[][] innerGenerate(char[][] map)
590    {
591        OrderedSet<Coord> doorways;
592        OrderedSet<Coord> hazards = new OrderedSet<>();
593        Coord temp;
594        boolean doubleDoors = false;
595        int floorCount = DungeonUtility.countCells(map, '.'),
596                doorFill = 0,
597                waterFill = 0,
598                grassFill = 0,
599                trapFill = 0,
600                boulderFill = 0,
601                islandSpacing = 0;
602        if(fx.containsKey(FillEffect.DOORS))
603        {
604            doorFill = fx.get(FillEffect.DOORS);
605            if(doorFill < 0)
606            {
607                doubleDoors = true;
608                doorFill *= -1;
609            }
610        }
611        if(fx.containsKey(FillEffect.GRASS)) {
612            grassFill = fx.get(FillEffect.GRASS);
613        }
614        if(fx.containsKey(FillEffect.WATER)) {
615            waterFill = fx.get(FillEffect.WATER);
616        }
617        if(fx.containsKey(FillEffect.BOULDERS)) {
618            boulderFill = fx.get(FillEffect.BOULDERS) * floorCount / 100;
619        }
620        if(fx.containsKey(FillEffect.ISLANDS)) {
621            islandSpacing = fx.get(FillEffect.ISLANDS);
622        }
623        if(fx.containsKey(FillEffect.TRAPS)) {
624            trapFill = fx.get(FillEffect.TRAPS);
625        }
626
627        doorways = viableDoorways(doubleDoors, map);
628
629        OrderedSet<Coord> obstacles = new OrderedSet<>(doorways.size() * doorFill / 100);
630        if(doorFill > 0)
631        {
632            int total = doorways.size() * doorFill / 100;
633
634            BigLoop:
635            for(int i = 0; i < total; i++)
636            {
637                Coord entry = rng.getRandomElement(doorways);
638                if(map[entry.x][entry.y] == '<' || map[entry.x][entry.y] == '>')
639                    continue;
640                if(map[entry.x - 1][entry.y] != '#' && map[entry.x + 1][entry.y] != '#')
641                {
642                    map[entry.x][entry.y] = '+';
643                }
644                else {
645                    map[entry.x][entry.y] = '/';
646                }
647                obstacles.add(entry);
648                Coord[] adj = new Coord[]{Coord.get(entry.x + 1, entry.y), Coord.get(entry.x - 1, entry.y),
649                        Coord.get(entry.x, entry.y + 1), Coord.get(entry.x, entry.y - 1)};
650                for(Coord near : adj) {
651                    if (doorways.contains(near)) {
652                        map[near.x][near.y] = '#';
653                        obstacles.add(near);
654                        doorways.remove(near);
655                        i++;
656                        doorways.remove(entry);
657                        continue BigLoop;
658                    }
659                }
660                doorways.remove(entry);
661            }
662        }
663        if (boulderFill > 0.0) {
664            /*
665            short[] floor = pack(map, '.');
666            short[] viable = retract(floor, 1, width, height, true);
667            ArrayList<Coord> boulders = randomPortion(viable, boulderFill, rng);
668            for (Coord boulder : boulders) {
669                map[boulder.x][boulder.y] = '#';
670            }
671            */
672            Coord[] boulders = new GreasedRegion(map, '.').retract8way(1).randomPortion(rng, boulderFill);
673            Coord t;
674            for (int i = 0; i < boulders.length; i++) {
675                t = boulders[i];
676                map[t.x][t.y] = '#';
677            }
678        }
679
680
681        if(trapFill > 0) {
682            for (int x = 1; x < map.length - 1; x++) {
683                for (int y = 1; y < map[x].length - 1; y++) {
684                    temp = Coord.get(x, y);
685                    if (map[x][y] == '.' && !obstacles.contains(temp)) {
686                        int ctr = 0;
687                        if (map[x + 1][y] != '#') ++ctr;
688                        if (map[x - 1][y] != '#') ++ctr;
689                        if (map[x][y + 1] != '#') ++ctr;
690                        if (map[x][y - 1] != '#') ++ctr;
691                        if (map[x + 1][y + 1] != '#') ++ctr;
692                        if (map[x - 1][y + 1] != '#') ++ctr;
693                        if (map[x + 1][y - 1] != '#') ++ctr;
694                        if (map[x - 1][y - 1] != '#') ++ctr;
695                        if (ctr >= 5) hazards.add(Coord.get(x, y));
696                    }
697                }
698            }
699        }
700        GreasedRegion floors = new GreasedRegion(map, '.'), working = new GreasedRegion(width, height);
701        floorCount = floors.size();
702        float waterRate = waterFill / 100.0f, grassRate = grassFill / 100.0f;
703        if(waterRate + grassRate > 1.0f)
704        {
705            waterRate /= (waterFill + grassFill) / 100.0f;
706            grassRate /= (waterFill + grassFill) / 100.0f;
707        }
708        int targetWater = Math.round(floorCount * waterRate),
709                targetGrass = Math.round(floorCount * grassRate),
710                sizeWaterPools = targetWater / rng.between(3, 6),
711                sizeGrassPools = targetGrass / rng.between(2, 5);
712
713        Coord[] scatter;
714        int remainingWater = targetWater, remainingGrass = targetGrass;
715        if(targetWater > 0) {
716            scatter = floors.quasiRandomSeparated(1.0 / 7.0);
717            rng.shuffleInPlace(scatter);
718            GreasedRegion allWater = new GreasedRegion(width, height);
719            for (int i = 0; i < scatter.length; i++) {
720                if (remainingWater > 5)
721                {
722                    if(!floors.contains(scatter[i]))
723                        continue;
724                    working.empty().insert(scatter[i]).spill(floors, rng.between(4, Math.min(remainingWater, sizeWaterPools)), rng);
725
726                    floors.andNot(working);
727                    remainingWater -= working.size();
728                    allWater.addAll(working);
729                } else
730                    break;
731            }
732
733            for (Coord pt : allWater) {
734                hazards.remove(pt);
735                //obstacles.add(pt);
736                if (map[pt.x][pt.y] != '<' && map[pt.x][pt.y] != '>')
737                    map[pt.x][pt.y] = '~';
738            }
739            for (Coord pt : allWater) {
740                if (map[pt.x][pt.y] != '<' && map[pt.x][pt.y] != '>' &&
741                        (map[pt.x - 1][pt.y] == '.' || map[pt.x + 1][pt.y] == '.' ||
742                                map[pt.x][pt.y - 1] == '.' || map[pt.x][pt.y + 1] == '.'))
743                    map[pt.x][pt.y] = ',';
744            }
745        }
746        if(targetGrass > 0) {
747            scatter = floors.quasiRandomSeparated(1.03/6.7);
748            rng.shuffleInPlace(scatter);
749            for (int i = 0; i < scatter.length; i++) {
750                if (remainingGrass > 5) //remainingGrass >= targetGrass * 0.02 &&
751                {
752                    working.empty().insert(scatter[i]).spill(floors, rng.between(4, Math.min(remainingGrass, sizeGrassPools)), rng);
753                    if (working.isEmpty())
754                        continue;
755                    floors.andNot(working);
756                    remainingGrass -= working.size();
757                    map = working.inverseMask(map, '"');
758                } else
759                    break;
760            }
761        }
762
763        if(islandSpacing > 1 && targetWater > 0) {
764            ArrayList<Coord> islands = PoissonDisk.sampleMap(map, 1f * islandSpacing, rng, '#', '.', '"', '+', '/', '^', '<', '>');
765            for (Coord c : islands) {
766                map[c.x][c.y] = '.';
767                if (map[c.x - 1][c.y] != '#' && map[c.x - 1][c.y] != '<' && map[c.x - 1][c.y] != '>')
768                    map[c.x - 1][c.y] = ',';
769                if (map[c.x + 1][c.y] != '#' && map[c.x + 1][c.y] != '<' && map[c.x + 1][c.y] != '>')
770                    map[c.x + 1][c.y] = ',';
771                if (map[c.x][c.y - 1] != '#' && map[c.x][c.y - 1] != '<' && map[c.x][c.y - 1] != '>')
772                    map[c.x][c.y - 1] = ',';
773                if (map[c.x][c.y + 1] != '#' && map[c.x][c.y + 1] != '<' && map[c.x][c.y + 1] != '>')
774                    map[c.x][c.y + 1] = ',';
775            }
776        }
777
778        if(trapFill > 0)
779        {
780            int total = hazards.size() * trapFill / 100;
781
782            for(int i = 0; i < total; i++)
783            {
784                Coord entry = rng.getRandomElement(hazards);
785                if(map[entry.x][entry.y] == '<' || map[entry.x][entry.y] == '<')
786                    continue;
787                map[entry.x][entry.y] = '^';
788                hazards.remove(entry);
789            }
790        }
791
792        dungeon = map;
793        return map;
794
795    }

800     * will be used to generate the first dungeon if none has been made yet). You can pass the long this returns to
801     * the setState() method on this class' rng field, which assuming all other calls to generate a dungeon are
802     * identical, will ensure generate() or generateRespectingStairs() will produce the same dungeon output as the
803     * dungeon originally generated with the seed this returned.
804     * <br>

806     * handles some complexities of when the state is actually used to generate a dungeon; since StatefulRNG objects can
807     * be shared between different classes that use random numbers, the state could change between when you call
808     * getState() and when this class generates a dungeon. Using getRebuildSeed() eliminates that confusion.
809     * @return a seed as a long that can be passed to setState() on this class' rng field to recreate a dungeon
810     */

812        return rebuildSeed;

817     *
818     * @return a printable string version of the latest generated dungeon.
819     */
820    @Override
821        public String toString() {
822        char[][] trans = new char[height][width];
823        for (int x = 0; x < width; x++) {
824            for (int y = 0; y < height; y++) {
825                trans[y][x] = dungeon[x][y];
826            }
827        }
828        StringBuffer sb = new StringBuffer();
829        for (int row = 0; row < height; row++) {
830            sb.append(trans[row]);
831            sb.append('\n');
832        }
833        return sb.toString();
834    }
