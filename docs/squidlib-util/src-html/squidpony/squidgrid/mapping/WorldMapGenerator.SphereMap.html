009/**
010 * Can be used to generate world maps with a wide variety of data, starting with height, temperature and moisture.
011 * From there, you can determine biome information in as much detail as your game needs, with a default implementation
012 * available that assigns a single biome to each cell based on heat/moisture. The maps this produces are valid
013 * for spherical or toroidal world projections, and will wrap from edge to opposite edge seamlessly thanks to a
014 * technique from the Accidental Noise Library ( https://www.gamedev.net/blog/33/entry-2138456-seamless-noise/ ) that
015 * involves getting a 2D slice of 4D Simplex noise. Because of how Simplex noise works, this also allows extremely high
016 * zoom levels as long as certain parameters are within reason. You can access the height map with the
017 * {@link #heightData} field, the heat map with the {@link #heatData} field, the moisture map with the
018 * {@link #moistureData} field, and a special map that stores ints representing the codes for various ranges of
019 * elevation (0 to 8 inclusive, with 0 the deepest ocean and 8 the highest mountains) with {@link #heightCodeData}. The
020 * last map should be noted as being the simplest way to find what is land and what is water; any height code 4 or
021 * greater is land, and any height code 3 or less is water. This can produce rivers, and keeps that information in a
022 * GreasedRegion (alongside a GreasedRegion containing lake positions) instead of in the other map data. This class does
023 * not use Coord at all, but if you want maps with width and/or height greater than 256, and you want to use the river
024 * or lake data as a Collection of Coord, then you should call {@link squidpony.squidmath.Coord#expandPoolTo(int, int)}
025 * with your width and height so the Coords remain safely pooled. If you're fine with keeping rivers and lakes as
026 * GreasedRegions and not requesting Coord values from them, then you don't need to do anything with Coord. Certain
027 * parts of this class are not necessary to generate, just in case you want river-less maps or something similar;
028 * setting {@link #generateRivers} to false will disable river generation (it defaults to true).
029 * <br>
030 * The main trade-off this makes to obtain better quality is reduced speed; generating a 512x512 map on a circa-2016
031 * laptop (i7-6700HQ processor at 2.6 GHz) takes about 1 second (about 1.15 seconds for an un-zoomed map, 0.95 or so
032 * seconds to increase zoom at double resolution). If you don't need a 512x512 map, this takes commensurately less time
033 * to generate less grid cells, with 64x64 maps generating faster than they can be accurately seen on the same hardware.
034 * River positions are produced using a different method, and do not involve the Simplex noise parts other than using
035 * the height map to determine flow. Zooming with rivers is tricky, and generally requires starting from the outermost
036 * zoom level and progressively enlarging and adding detail to all rivers as zoom increases on specified points.
037 */

147    /**
148     * Generates a world using the specified RNG state as a long, with specific water and cooling modifiers that affect
149     * the land-water ratio and the average temperature, respectively.
150     * The worlds this produces will always have width and height as specified in the constructor (default 256x256).
151     * You can call {@link #zoomIn(int, int, int)} to double the resolution and center on the specified area, but the width
152     * and height of the 2D arrays this changed, such as {@link #heightData} and {@link #moistureData} will be the same.
153     * @param waterMod should be between 0.85 and 1.2; a random value will be used if this is negative
154     * @param coolMod should be between 0.85 and 1.4; a random value will be used if this is negative
155     * @param state the state to give this generator's RNG; if the same as the last call, this will reuse data
156     */
157    public void generate(double waterMod, double coolMod, long state)
158    {
159        if(cachedState != state || waterMod != waterModifier || coolMod != coolingModifier)
160        {
161            seed = state;
162            zoom = 0;
163            startCacheX.clear();
164            startCacheY.clear();
165            startCacheX.add(0);
166            startCacheY.add(0);
167
168        }
169        regenerate(startCacheX.peek(), startCacheY.peek(),
170                (width >> zoom), (height >> zoom), waterMod, coolMod, state);
171    }

183    /**
184     * Halves the resolution of the map and doubles the area it covers repeatedly, halving {@code zoomAmount} times; the
185     * 2D arrays this uses keep their sizes. This version of zoomOut allows you to specify where the zoom should be
186     * centered, using the current coordinates (if the map size is 256x256, then coordinates should be between 0 and
187     * 255, and will refer to the currently used area and not necessarily the full world size).
188     * <br>
189     * Only has an effect if you have previously zoomed in using {@link #zoomIn(int, int, int)} or its overload.
190     * @param zoomCenterX the center X position to zoom out from; if too close to an edge, this will stop moving before it would extend past an edge
191     * @param zoomCenterY the center Y position to zoom out from; if too close to an edge, this will stop moving before it would extend past an edge
192     */
193    public void zoomOut(int zoomAmount, int zoomCenterX, int zoomCenterY)
194    {
195        if(zoomAmount == 0) return;
196        if(zoomAmount < 0) {
197            zoomIn(-zoomAmount, zoomCenterX, zoomCenterY);
198            return;
199        }
200        if(zoom > 0)
201        {
202            if(seed != cachedState)
203            {
204                generate(rng.nextLong());
205            }
206
207            zoom -= zoomAmount;
208            startCacheX.pop();
209            startCacheY.pop();
210            startCacheX.add(Math.min(Math.max(startCacheX.pop() + (zoomCenterX >> zoom + 1) - (width >> zoom + 2),
211                    0), width - (width >> zoom)));
212            startCacheY.add(Math.min(Math.max(startCacheY.pop() + (zoomCenterY >> zoom + 1) - (height >> zoom + 2),
213                    0), height - (height >> zoom)));
214            regenerate(startCacheX.peek(), startCacheY.peek(),width >> zoom, height >> zoom,
215                    waterModifier, coolingModifier, cachedState);
216            rng.setState(cachedState);
217        }
218
219    }

221     * Doubles the resolution of the map and halves the area it covers; the 2D arrays this uses keep their sizes. This
222     * version of zoomIn always zooms in to the center of the currently used area.
223     * <br>
224     * Although there is no technical restriction on maximum zoom, zooming in more than 5 times (64x scale or greater)

232    /**
233     * Doubles the resolution of the map and halves the area it covers repeatedly, doubling {@code zoomAmount} times;
234     * the 2D arrays this uses keep their sizes. This version of zoomIn allows you to specify where the zoom should be
235     * centered, using the current coordinates (if the map size is 256x256, then coordinates should be between 0 and
236     * 255, and will refer to the currently used area and not necessarily the full world size).
237     * <br>
238     * Although there is no technical restriction on maximum zoom, zooming in more than 5 times (64x scale or greater)
239     * will make the map appear somewhat less realistic due to rounded shapes appearing more bubble-like and less like a
240     * normal landscape.
241     * @param zoomCenterX the center X position to zoom in to; if too close to an edge, this will stop moving before it would extend past an edge
242     * @param zoomCenterY the center Y position to zoom in to; if too close to an edge, this will stop moving before it would extend past an edge
243     */
244    public void zoomIn(int zoomAmount, int zoomCenterX, int zoomCenterY)
245    {
246        if(zoomAmount == 0) return;
247        if(zoomAmount < 0)
248        {
249            zoomOut(-zoomAmount, zoomCenterX, zoomCenterY);
250            return;
251        }
252        if(seed != cachedState)
253        {
254            generate(rng.nextLong());
255        }
256        zoom += zoomAmount;
257        if(startCacheX.isEmpty())
258        {
259            startCacheX.add(0);
260            startCacheY.add(0);
261        }
262        else {
263            startCacheX.add(Math.min(Math.max(startCacheX.peek() + (zoomCenterX >> zoom - 1) - (width >> zoom + 1),
264                    0), width - (width >> zoom)));
265            startCacheY.add(Math.min(Math.max(startCacheY.peek() + (zoomCenterY >> zoom - 1) - (height >> zoom + 1),
266                    0), height - (height >> zoom)));
267        }
268        regenerate(startCacheX.peek(), startCacheY.peek(),width >> zoom, height >> zoom,
269                waterModifier, coolingModifier, cachedState);
270        rng.setState(cachedState);
271    }

282        if(high < shallowWaterUpper)

288        if(high < grassUpper)

292        if(high < rockUpper)

310    private static final Direction[] reuse = new Direction[6];
311    private void appendDirToShuffle(RNG rng) {
312        rng.randomPortion(Direction.CARDINALS, reuse);
313        reuse[rng.next(2)] = Direction.DIAGONALS[rng.next(2)];
314        reuse[4] = Direction.DIAGONALS[rng.next(2)];
315        reuse[5] = Direction.OUTWARDS[rng.next(3)];

318    protected void addRivers()
319    {
320        landData.refill(heightCodeData, 4, 999);
321        long rebuildState = rng.nextLong();
322        //workingData.allOn();
323                //.empty().insertRectangle(8, 8, width - 16, height - 16);
324        riverData.empty().refill(heightCodeData, 6, 100);
325        riverData.quasiRandomRegion(0.0066);
326        int[] starts = riverData.asTightEncoded();
327        int len = starts.length, currentPos, choice, adjX, adjY, currX, currY, tcx, tcy, stx, sty, sbx, sby;
328        riverData.clear();
329        lakeData.clear();
330        PER_RIVER:
331        for (int i = 0; i < len; i++) {
332            workingData.clear();
333            currentPos = starts[i];
334            stx = tcx = currX = decodeX(currentPos);
335            sty = tcy = currY = decodeY(currentPos);
336            while (true) {
337
338                double best = 999999;
339                choice = -1;
340                appendDirToShuffle(rng);
341
342                for (int d = 0; d < 5; d++) {
343                    adjX = wrapX(currX + reuse[d].deltaX);
344                    /*
345                    if (adjX < 0 || adjX >= width)
346                    {
347                        if(rng.next(4) == 0)
348                            riverData.or(workingData);
349                        continue PER_RIVER;
350                    }*/
351                    adjY = wrapY(currY + reuse[d].deltaY);
352                    if (heightData[adjX][adjY] < best && !workingData.contains(adjX, adjY)) {
353                        best = heightData[adjX][adjY];
354                        choice = d;
355                        tcx = adjX;
356                        tcy = adjY;
357                    }
358                }
359                currX = tcx;
360                currY = tcy;
361                if (best >= heightData[stx][sty]) {
362                    tcx = rng.next(2);
363                    adjX = wrapX(currX + ((tcx & 1) << 1) - 1);
364                    adjY = wrapY(currY + (tcx & 2) - 1);
365                    lakeData.insert(currX, currY);
366                    lakeData.insert(wrapX(currX+1), currY);
367                    lakeData.insert(wrapX(currX-1), currY);
368                    lakeData.insert(currX, wrapY(currY+1));
369                    lakeData.insert(currX, wrapY(currY-1));
370
371                    if(heightCodeData[adjX][adjY] <= 3) {
372                        riverData.or(workingData);
373                        continue PER_RIVER;
374                    }
375                    else if((heightData[adjX][adjY] -= 0.0002) < 0.0) {
376                        if (rng.next(3) == 0)
377                            riverData.or(workingData);
378                        continue PER_RIVER;
379                    }
380                    tcx = rng.next(2);
381                    adjX = wrapX(currX + ((tcx & 1) << 1) - 1);
382                    adjY = wrapY(currY + (tcx & 2) - 1);
383                    if(heightCodeData[adjX][adjY] <= 3) {
384                        riverData.or(workingData);
385                        continue PER_RIVER;
386                    }
387                    else if((heightData[adjX][adjY] -= 0.0002) < 0.0) {
388                        if (rng.next(3) == 0)
389                            riverData.or(workingData);
390                        continue PER_RIVER;
391                    }
392                }
393                if(choice != -1 && reuse[choice].isDiagonal())
394                {
395                    tcx = wrapX(currX - reuse[choice].deltaX);
396                    tcy = wrapY(currY - reuse[choice].deltaY);
397                    if(heightData[tcx][currY] <= heightData[currX][tcy] && !workingData.contains(tcx, currY))
398                    {
399                        if(heightCodeData[tcx][currY] < 3 || riverData.contains(tcx, currY))
400                        {
401                            riverData.or(workingData);
402                            continue PER_RIVER;
403                        }
404                        workingData.insert(tcx, currY);
405                    }
406                    else if(!workingData.contains(currX, tcy))
407                    {
408                        if(heightCodeData[currX][tcy] < 3 || riverData.contains(currX, tcy))
409                        {
410                            riverData.or(workingData);
411                            continue PER_RIVER;
412                        }
413                        workingData.insert(currX, tcy);
414
415                    }
416                }
417                if(heightCodeData[currX][currY] < 3 || riverData.contains(currX, currY))
418                {
419                    riverData.or(workingData);
420                    continue PER_RIVER;
421                }
422                workingData.insert(currX, currY);
423            }
424        }
425
426        GreasedRegion tempData = new GreasedRegion(width, height);
427        int riverCount = riverData.size() >> 4, currentMax = riverCount >> 3, idx = 0, prevChoice;
428        for (int h = 5; h < 9; h++) { //, currentMax += riverCount / 18
429            workingData.empty().refill(heightCodeData, h).and(riverData);
430            RIVER:
431            for (int j = 0; j < currentMax && idx < riverCount; j++) {
432                double vdc = VanDerCorputQRNG.weakDetermine(idx++), best = -999999;
433                currentPos = workingData.atFractionTight(vdc);
434                if(currentPos < 0)
435                    break;
436                stx = sbx = tcx = currX = decodeX(currentPos);
437                sty = sby = tcy = currY = decodeY(currentPos);
438                appendDirToShuffle(rng);
439                choice = -1;
440                prevChoice = -1;
441                for (int d = 0; d < 5; d++) {
442                    adjX = wrapX(currX + reuse[d].deltaX);
443                    adjY = wrapY(currY + reuse[d].deltaY);
444                    if (heightData[adjX][adjY] > best) {
445                        best = heightData[adjX][adjY];
446                        prevChoice = choice;
447                        choice = d;
448                        sbx = tcx;
449                        sby = tcy;
450                        tcx = adjX;
451                        tcy = adjY;
452                    }
453                }
454                currX = sbx;
455                currY = sby;
456                if (prevChoice != -1 && heightCodeData[currX][currY] >= 4) {
457                    if (reuse[prevChoice].isDiagonal()) {
458                        tcx = wrapX(currX - reuse[prevChoice].deltaX);
459                        tcy = wrapY(currY - reuse[prevChoice].deltaY);
460                        if (heightData[tcx][currY] <= heightData[currX][tcy]) {
461                            if(heightCodeData[tcx][currY] < 3)
462                            {
463                                riverData.or(tempData);
464                                continue;
465                            }
466                            tempData.insert(tcx, currY);
467                        }
468                        else
469                        {
470                            if(heightCodeData[currX][tcy] < 3)
471                            {
472                                riverData.or(tempData);
473                                continue;
474                            }
475                            tempData.insert(currX, tcy);
476                        }
477                    }
478                    if(heightCodeData[currX][currY] < 3)
479                    {
480                        riverData.or(tempData);
481                        continue;
482                    }
483                    tempData.insert(currX, currY);
484                }
485
486                while (true) {

488                    appendDirToShuffle(rng);
489                    choice = -1;
490                    for (int d = 0; d < 6; d++) {
491                        adjX = wrapX(currX + reuse[d].deltaX);
492                        adjY = wrapY(currY + reuse[d].deltaY);
493                        if (heightData[adjX][adjY] > best && !riverData.contains(adjX, adjY)) {

495                            choice = d;
496                            sbx = adjX;
497                            sby = adjY;
498                        }
499                    }
500                    currX = sbx;
501                    currY = sby;
502                    if (choice != -1) {
503                        if (reuse[choice].isDiagonal()) {
504                            tcx = wrapX(currX - reuse[choice].deltaX);
505                            tcy = wrapY(currY - reuse[choice].deltaY);
506                            if (heightData[tcx][currY] <= heightData[currX][tcy]) {
507                                if(heightCodeData[tcx][currY] < 3)
508                                {
509                                    riverData.or(tempData);
510                                    continue RIVER;
511                                }
512                                tempData.insert(tcx, currY);
513                            }
514                            else
515                            {
516                                if(heightCodeData[currX][tcy] < 3)
517                                {
518                                    riverData.or(tempData);
519                                    continue RIVER;
520                                }
521                                tempData.insert(currX, tcy);
522                            }
523                        }
524                        if(heightCodeData[currX][currY] < 3)
525                        {
526                            riverData.or(tempData);
527                            continue RIVER;
528                        }

530                    }
531                    else
532                    {
533                        riverData.or(tempData);

536                    }

541                            continue RIVER;

544                        sbx &= sbx >>> 4;
545                        if ((sbx & 1) == 0)
546                            lakeData.insert(wrapX(currX + 1), currY);
547                        if ((sbx & 2) == 0)
548                            lakeData.insert(wrapX(currX - 1), currY);
549                        if ((sbx & 4) == 0)
550                            lakeData.insert(currX, wrapY(currY + 1));
551                        if ((sbx & 8) == 0)
552                            lakeData.insert(currX, wrapY(currY - 1));
553                        sbx = rng.next(2);
554                        lakeData.insert(wrapX(currX + (-(sbx & 1) | 1)), wrapY(currY + ((sbx & 2) - 1))); // random diagonal
555                        lakeData.insert(currX, wrapY(currY + ((sbx & 2) - 1))); // ortho next to random diagonal
556                        lakeData.insert(wrapX(currX + (-(sbx & 1) | 1)), currY); // ortho next to random diagonal
557
558                        continue RIVER;
559                    }
560                }
561            }
562
563        }
564
565        rng.setState(rebuildState);

645         * Analyzes the last world produced by the given WorldMapGenerator and uses all of its generated information to
646         * assign biome codes for each cell (along with heat and moisture codes). After calling this, biome codes can be
647         * taken from {@link #biomeCodeData} and used as indices into {@link #biomeTable} or a custom biome table.
648         * @param world a WorldMapGenerator that should have generated at least one map; it may be at any zoom
649         */
650        public void makeBiomes(WorldMapGenerator world) {
651            if(world == null || world.width <= 0 || world.height <= 0)
652                return;
653            if(heatCodeData == null || (heatCodeData.length != world.width || heatCodeData[0].length != world.height))
654                heatCodeData = new int[world.width][world.height];
655            if(moistureCodeData == null || (moistureCodeData.length != world.width || moistureCodeData[0].length != world.height))
656                moistureCodeData = new int[world.width][world.height];
657            if(biomeCodeData == null || (biomeCodeData.length != world.width || biomeCodeData[0].length != world.height))
658                biomeCodeData = new int[world.width][world.height];
659            final double i_hot = (world.maxHeat == world.minHeat) ? 1.0 : 1.0 / (world.maxHeat - world.minHeat);
660            for (int x = 0; x < world.width; x++) {
661                for (int y = 0; y < world.height; y++) {
662                    final double hot = (world.heatData[x][y] - world.minHeat) * i_hot, moist = world.moistureData[x][y];
663                    final int heightCode = world.heightCodeData[x][y];
664                    int hc, mc;
665                    boolean isLake = world.generateRivers && world.partialLakeData.contains(x, y) && heightCode >= 4,
666                            isRiver = world.generateRivers && world.partialRiverData.contains(x, y) && heightCode >= 4;
667                    if (moist > wetterValueUpper) {
668                        mc = 5;
669                    } else if (moist > wetValueUpper) {
670                        mc = 4;
671                    } else if (moist > dryValueUpper) {
672                        mc = 3;
673                    } else if (moist > drierValueUpper) {
674                        mc = 2;
675                    } else if (moist > driestValueUpper) {
676                        mc = 1;
677                    } else {
678                        mc = 0;
679                    }
680
681                    if (hot > warmerValueUpper) {
682                        hc = 5;
683                    } else if (hot > warmValueUpper) {
684                        hc = 4;
685                    } else if (hot > coldValueUpper) {
686                        hc = 3;
687                    } else if (hot > colderValueUpper) {
688                        hc = 2;
689                    } else if (hot > coldestValueUpper) {
690                        hc = 1;
691                    } else {
692                        hc = 0;
693                    }
694
695                    heatCodeData[x][y] = hc;
696                    moistureCodeData[x][y] = mc;
697                    biomeCodeData[x][y] = isLake ? hc + 48 : (isRiver ? hc + 42 : ((heightCode == 4) ? hc + 36 : hc + mc * 6));
698                }
699            }
700        }

813            if(cachedState != state || waterMod != waterModifier || coolMod != coolingModifier)
814            {
815                minHeight = Double.POSITIVE_INFINITY;
816                maxHeight = Double.NEGATIVE_INFINITY;
817                minHeat0 = Double.POSITIVE_INFINITY;
818                maxHeat0 = Double.NEGATIVE_INFINITY;

831            int seedA = rng.nextInt(), seedB = rng.nextInt(), seedC = rng.nextInt(), t;
832
833            waterModifier = (waterMod <= 0) ? rng.nextDouble(0.25) + 0.89 : waterMod;
834            coolingModifier = (coolMod <= 0) ? rng.nextDouble(0.45) * (rng.nextDouble()-0.5) + 1.1 : coolMod;
835
836            double p, q,
837                    ps, pc,
838                    qs, qc,
839                    h, temp,
840                    i_w = 6.283185307179586 / width, i_h = 6.283185307179586 / height,
841                    xPos = startX, yPos = startY, i_uw = usedWidth / (double)width, i_uh = usedHeight / (double)height;
842            double[] trigTable = new double[width << 1];
843            for (int x = 0; x < width; x++, xPos += i_uw) {
844                p = xPos * i_w;
845                trigTable[x<<1]   = Math.sin(p);
846                trigTable[x<<1|1] = Math.cos(p);

848            for (int y = 0; y < height; y++, yPos += i_uh) {
849                q = yPos * i_h;
850                qs = Math.sin(q);
851                qc = Math.cos(q);

856                                    terrainRidged.getNoiseWithSeed(pc, ps, qc, qs, seedA + seedB),

882
883            }

886                    wetDiff = 1.0 / (maxWet0 - minWet0),

909                        case 3:

911                            hMod = 0.2;
912                            break;
913                        case 6:
914                            h = -0.1 * (h - forestLower - 0.08);
915                            break;
916                        case 7:
917                            h *= -0.25;
918                            break;
919                        case 8:
920                            h *= -0.4;
921                            break;
922                        default:
923                            h *= 0.05;
924                    }

926                    if (fresh) {
927                        ps = Math.min(ps, h); //minHeat0
928                        pc = Math.max(pc, h); //maxHeat0

936            }

947                    moistureData[x][y] = (temp = (moistureData[x][y] - minWet0) * wetDiff);
948                    if (fresh) {
949                        qs = Math.min(qs, h);
950                        qc = Math.max(qc, h);
951                        ps = Math.min(ps, temp);
952                        pc = Math.max(pc, temp);
953                    }
954                }
955            }
956            if(fresh)
957            {
958                minHeat = qs;
959                maxHeat = qc;
960                minWet = ps;
961                maxWet = pc;
962            }

985                            partialLakeData.zoom(stx, sty).connect8way().thin();
986                            partialLakeData.or(workingData.remake(partialLakeData).fringe().quasiRandomRegion(0.7));

988                    }
