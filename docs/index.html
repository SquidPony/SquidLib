Welcome to SquidLib's Documentation.

The main page, including source code, is on GitHub. All of the javadocs for the latest (built) version of code are in subdirectories of this site. You can also download javadocs in a .jar for released versions of SquidLib using Maven, Gradle, or simply by going to the Maven Central page for SquidLib and downloading the jars you want. As for javadocs...

  • For the core of SquidLib, which can be used from any Java project, go here for squidlib-util.
  • For the text rendering part of SquidLib, which uses libGDX, go here for squidlib.
  • For the optional functionality of SquidLib (like handling saved files), which depends on libGDX only for its utilities, go here for squidlib-extra.
